## 📖 Overview
Sentinel Scout is a free, zero-config CLI tool that scans Python codebases for dangerous execution vectors such as eval, exec, and subprocess, which can lead to devastating security breaches. It uses static analysis to identify these vulnerabilities, providing a structured Risk Manifest to help developers and founders secure their codebases. This tool is designed to be easy to use, with no complex setup or financial investment required, making it accessible to anyone who wants to protect their Python code. By using Sentinel Scout, developers can instantly identify potential security risks and take steps to mitigate them.
